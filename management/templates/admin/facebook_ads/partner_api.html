    Alur Opsi 1 — Via API (otomatis)

    1. Admin HRIS klik ikon pesawat di halaman Account Facebook Ads.
    2. Jika FACEBOOK_PARTNER_FB_ACCESS_TOKEN sudah diset di .env, HRIS langsung simpan token otomatis (tanpa curl manual).
    3. Opsional: HRIS kirim webhook ke server Partner BM eksternal.
    4. Alternatif manual: Partner POST token EAAG... ke endpoint submit di bawah.
    Opsi termudah (server HRIS = partner): set token Facebook di .env, restart gunicorn, klik ikon pesawat. {% if partner_auto_token_configured %} Auto-submit: aktif {% else %} Auto-submit: belum diset {% endif %}
    Jangan paste request_token HRIS ke kolom Access Token di Edit Account. Kolom itu hanya untuk token Facebook (EAAG...).
    1. Konfigurasi HRIS (.env)
    FACEBOOK_PARTNER_FB_ACCESS_TOKEN=EAAG...   # wajib untuk auto-submit saat klik pesawat
FACEBOOK_PARTNER_API_KEY={{ partner_api_key_hint }}
FACEBOOK_PARTNER_WEBHOOK_URL={{ partner_webhook_url_hint }}
FACEBOOK_PARTNER_WEBHOOK_SECRET={{ partner_webhook_secret_hint }}

    Setelah edit .env: sudo systemctl restart hris-gunicorn

    2. Webhook HRIS → Partner BM

    HRIS POST ke FACEBOOK_PARTNER_WEBHOOK_URL dengan body:

    {
  "event": "facebook_token_request",
  "request_token": "...",
  "submit_url": "{{ submit_url }}",
  "expires_at": 1234567890,
  "account_id": "act_1026837102798159",
  "account_name": "Hendi Budianto",
  "app_id": "1482344110223326",
  "required_scopes": ["ads_read", "ads_management", "business_management"]
}

    Header opsional: X-Webhook-Secret: {{ partner_webhook_secret_hint }}

    3. Partner BM → HRIS (submit token)

    Endpoint: {{ submit_url }}

    curl -X POST "{{ submit_url }}" \
  -H "Content-Type: application/json" \
  -H "X-API-Key: {{ partner_api_key_hint }}" \
  -d '{
    "request_token": "REQUEST_TOKEN_DARI_WEBHOOK_HRIS",
    "access_token": "EAAG...TOKEN_DARI_GRAPH_API_EXPLORER",
    "submitted_by": "partner-bm"
  }'
    4. Sample webhook server Partner BM

    Jalankan di server partner (repo HRIS, folder scripts/):

    export PARTNER_WEBHOOK_SECRET={{ partner_webhook_secret_hint }}
export HRIS_PARTNER_API_KEY={{ partner_api_key_hint }}
export PARTNER_FB_ACCESS_TOKEN=EAAG... # dari Graph API Explorer
python3 scripts/facebook_partner_webhook_server.py

    Lalu set FACEBOOK_PARTNER_WEBHOOK_URL ke URL publik partner, mis.:

    http://IP_SERVER_PARTNER:8787/hris/facebook-token-request
    5. Generate token Facebook (Partner BM)
    1. Buka Graph API Explorer
    2. App: Berita21 (1482344110223326)
    3. Permissions: ads_read, ads_management, business_management
    4. Generate Access Token → login akun yang punya akses ad account → Allow
    5. Copy token EAAG... → set PARTNER_FB_ACCESS_TOKEN atau kirim via curl di atas
    Kembali ke Account Facebook Ads
